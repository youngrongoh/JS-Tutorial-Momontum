# Stack
- HTML / CSS
- Vanila JS

# Features
- 실시간 시계
- 유저 이름을 저장하고 불러올 수 있음
- todo 추가 및 제거, 완료 항목 체크 기능
- todo 리스트를 저장하고 불러올 수 있음
- 위치기반으로 사용자의 날씨, 기온, 위치 정보를 표시
